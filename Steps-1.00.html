Instructions
1. Create a SQL Server database called "TVShows"

2. Update the "server" variable with your SQL Server Name for the following notebooks:
a. WS_Metacritic.ipynb
b. WS_RottenTomatoes.ipynb
c. API_IMDB.ipynb

(*The server variable was placed at the top of the jupyter page for your convience) 

3. Execute the notebooks mentioned in the above step.  These scripts are essentially your ETL.  If there are any libraries or modules that need to be installed, please do so (pip install).
   Once executed successfully, confirm the following SQL tables have been created in your TVShows SQL database:
a. dbo.API_LU_Genre
b. dbo.API_TV_Show_Top_Rated
c. dbo.WS_Metacritic
d. dbo.WS_RottenTomatoes
4. Execute the sql file - uspQuestion3.sql

5. Final step - Open Show_Suggester.ipynb
a. Like you did in step 2, update the "server" variable with your SQL Server Name.  Do a pip install for any missing libraries or                     modules. 
b. After updated the server name and executing (Ctrl + Enter), execute the 2nd cell.  This will hide the cells input.  This is done to                        give the Jupyter notebook a servey look without the clutter of the code.  
c. Fill out the survey.  FYI, question 3 is the only active question thus far.
